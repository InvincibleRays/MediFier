HELPING YOU MEDICALLY

GET NOTIFIED ON YOUR NEAREST GP AVAILABILITY,
USE OUR SEARCHBOT TO FIND THE NEAREST AVAILABLE MEDICAL SERVICE TO YOU

Features

This project and the website aims at presenting web scraped data

Chatbot

This is a medical search bot that asks for specific information before providing you with the 5 best suitable hospitals near you by distance

Search System

This is another search feature but this one specifically can be searched through the website itself, your UK postcode is to be searched. The task of this search is to display the 5 nearest General Practioners available near you

Mail System

A feature that would notify you through email if your nearest bookmarked practioner is available. This is an experimental feature that is supposed to be there in the final project but may not be.

About this Project

This was a website worked on as part of the HackBrunel V2 hackathon at Brunel University. The team members for this team are; Siddhant Sood, Suryaansh Maithani & Nilay Dedhia. We are all first year computer science students at Brunel University
